### Описание

Программа searcher предназначена для поиска файлов по индексу, построенному программой indexer. Рейтинг файлов строится по количеству вхождений слова из поискового запроса в файл, а также в зависимости от позиции слова в запросе - чем раньше слово идет в запросе, тем выше его приоритет.

### Создание файла конфигурации

Для работы с программой необходимо скопировать файл config_default.ini в config.ini, сохраняя формат файла, приведенный ниже. При этом:
- Строки комментариев в файле должны начинаться со знака #;
- Файл должен находиться в той же директории, что и программа;
- Значение "file_extensions" должно быть или пустое для индексирования всех типов файлов, или содержать расширения файлов разделенные запятой или запятой и пробелом; допустимые значения для расширений - символы латинского алфавита, цифры и нижнее подчеркивание;
- Значения параметров конфигурации должны быть в одинарных кавычках согласно примеру ниже; значения в двойных кавычках или без них могут привести к использованию параметров по умолчанию.

Пример файла конфигурации:
```

[db]
db_host = '127.0.0.1'
db_port = '5432'
db_name = 'indexer'
db_user = 'admin'
db_pass = 'admin123'
```

### Работа с программой

Для работы с программой необходима СУБД PostgreSQL, а также индекс, построенный программой indexer в соответствии с README.md данной программы.

При запуске программы и успешном подключении к БД поддерживается 2 функции:
- Поиск по запросу, доступен в основном окне программы;
- Просмотр индекса слов, доступен в меню "Просмотр слов и их частоты в БД".
